# ✈️ FlyWeather

**FlyWeather** is a Flutter-based mobile application that provides real-time weather information for airports around the world. It combines authentication with a simple, intuitive interface to explore airport-specific weather conditions.

---

## 🚀 Features

* 🔐 **Authentication System**

  * User login and sign-up
  * Basic session handling

* 🛫 **Airport Explorer**

  * Browse a list of airports
  * Clean and simple UI for navigation

* 🌦️ **Weather Details**

  * Real-time weather data for each airport
  * Includes:

    * Temperature (°C)
    * Humidity
    * Weather conditions (e.g. Clear, Rain, Clouds)
    * Additional atmospheric details

---

## 📱 Screens Overview

* Login / Sign-Up Screen
* Airport List Screen
* Airport Weather Details Screen

---

## ⚠️ Important Note About API Keys

The API keys included in this project are **not functional** (they are intentionally invalid/expired).

To run the app properly:

* Replace the existing API keys with your own
* Use any weather API provider (e.g. OpenWeather, WeatherAPI, etc.)
* Ensure your keys are correctly configured in the project files

## 🛠️ Tech Stack

* **Flutter** (Dart)

## ⚙️ Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/your-username/flyweather.git
cd flyweather
```

### 2. Install dependencies

```bash
flutter pub get
```

### 3. Configure API Keys

* Locate the API configuration file
* Replace placeholder/invalid keys with your own valid keys

### 4. Run the app

```bash
flutter run
```
